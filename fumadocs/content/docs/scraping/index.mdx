Het scraping-systeem verzamelt vacatures automatisch van geconfigureerde bronnen.

## Platforms

| Platform | Status | Type |
|----------|--------|------|
| Striive | Actief | Detachering |
| Opdrachtoverheid | Actief | Overheid |
| Flextender | Actief | Overheid |
| Indeed | Gepland | Algemeen |
| LinkedIn | Gepland | Professioneel |

## AI Verrijking

Na het scrapen worden nieuwe vacatures automatisch verrijkt door Gemini Flash:

| Veld | Beschrijving |
|------|-------------|
| `educationLevel` | HBO / WO / PhD |
| `workExperienceYears` | Geschatte vereiste ervaring |
| `workArrangement` | Remote / Hybrid / Onsite |
| `languages` | Vereiste talen |
| `durationMonths` | Contractduur |
| `extensionPossible` | Verlengbaar ja/nee |
| `descriptionSummary` | NL + EN samenvatting |
| `categories` | Classificaties |

## Foutafhandeling

<Mermaid chart={`
stateDiagram-v2
  [*] --> Active: Scraper geconfigureerd
  Active --> Running: Cron trigger
  Running --> Success: Vacatures gevonden
  Running --> Partial: Gedeeltelijk mislukt
  Running --> Failed: Volledig mislukt
  Success --> Active: Reset timer
  Partial --> Active: Log + waarschuwing
  Failed --> Retry: consecutiveFailures++
  Retry --> Running: Backoff wachttijd
  Retry --> Disabled: > 5 failures
  Disabled --> Active: Handmatige reset
`} />

## API

```bash
# Configuraties ophalen
GET /api/scraper-configuraties

# Handmatige scrape starten
POST /api/scrape/starten
{ "platform": "striive" }

# Systeem gezondheid
GET /api/gezondheid
```
