Welkom bij de documentatie van **Motian** — een AI-gestuurd recruitment platform gebouwd met Next.js, Drizzle ORM en Neon.

## Overzicht

Motian automatiseert het wervingsproces:

- **CV-analyse** — Upload en analyseer CV's met AI (Claude, GPT, Gemini)
- **Matching** — Vergelijk kandidaten met vacatures via gestructureerde scoring
- **Scraping** — Verzamel vacatures automatisch van externe bronnen
- **MCP-server** — Model Context Protocol integratie voor AI-assistenten
- **CLI** — Command-line interface voor directe platform-interactie

## Snel starten

```bash
# Installeer dependencies
pnpm install

# Start development server (poort 3001)
pnpm dev

# Voer tests uit
pnpm test
```
