## Duplicaat Detectie

Bij het opslaan van een geanalyseerd CV controleert het systeem:

1. **Email match** — Exact match op e-mailadres
2. **Naam fuzzy match** — Als email niet beschikbaar is, fuzzy vergelijking op naam

Bij een match wordt het bestaande profiel verrijkt in plaats van een duplicaat aangemaakt.

## API

```bash
# CV uploaden en analyseren
POST /api/cv-analyse
Content-Type: multipart/form-data
# Body: file (PDF/DOCX)

# CV bestand ophalen
GET /api/cv-file?url=<blob-url>
```
