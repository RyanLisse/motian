## CLI

### Gebruik

```bash
# CLI starten
pnpm cli

# Of via Just
just cli kandidaten:zoek --query "Java"
```

### Commando's (50+)

| Groep | Commando's | Voorbeeld |
|-------|-----------|-----------|
| **kandidaten** | zoek, detail, maak-aan, update, verwijder, notitie, auto-match | `kandidaten:zoek --skills "React,Node"` |
| **vacatures** | zoek, detail, update, verwijder, auto-match | `vacatures:zoek --platform striive` |
| **matches** | zoek, detail, maak-aan, goedkeuren, afwijzen, verwijder | `matches:goedkeuren --id <uuid>` |
| **sollicitaties** | zoek, detail, maak-aan, fase, stats | `sollicitaties:fase --id <uuid> --stage interview` |
| **interviews** | zoek, plan, update | `interviews:plan --type video` |
| **berichten** | zoek, stuur | `berichten:stuur --channel email` |
| **gdpr** | exporteer, wis, scrub, retentie | `gdpr:exporteer --candidate-id <uuid>` |
| **ops** | importeer, scoring-batch | `ops:importeer --platform striive` |

## MCP Server

De MCP server maakt het mogelijk om vanuit AI-tools (Claude Desktop, Cursor) het volledige platform te bedienen.

### Starten

```bash
# MCP server starten
pnpm mcp

# Via Just
just mcp

# Verificatie
just mcp-verify
```

### Voorbeeld MCP Interactie

Een AI-assistent kan via MCP:

```
Gebruiker: "Zoek kandidaten met React ervaring in Amsterdam"
  → MCP tool: zoek_kandidaten { query: "React", location: "Amsterdam" }

Gebruiker: "Match deze kandidaat met beschikbare vacatures"
  → MCP tool: auto_match_kandidaat { kandidaatId: "uuid" }

Gebruiker: "Keur de beste match goed"
  → MCP tool: keur_match_goed { matchId: "uuid" }
```

Alle MCP tools gebruiken Zod schema's die automatisch worden omgezet naar JSON Schema via `zod-to-json-schema`.
