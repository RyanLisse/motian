Motian biedt een CLI voor directe interactie met het platform.

## Gebruik

```bash
# CLI starten
pnpm cli

# Of direct via tsx
tsx src/cli/index.ts
```

## MCP Server

Het platform biedt ook een Model Context Protocol (MCP) server voor integratie met AI-assistenten zoals Claude.

```bash
# MCP server starten
pnpm mcp
```

De MCP server maakt het mogelijk om vanuit AI-tools kandidaten te zoeken, vacatures te beheren en matches te maken.
