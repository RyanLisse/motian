## Tech Stack

| Laag | Technologie | Doel |
|------|-------------|------|
| Frontend | Next.js 16 (App Router) | Server Components + Streaming |
| Database | Neon PostgreSQL + pgvector | Relationeel + vector similarity |
| AI | Vercel AI SDK | Multi-model (Gemini, OpenAI, Grok) |
| Achtergrondtaken | Trigger.dev v4 | Scraping, enrichment, batch |
| Monitoring | Sentry + PostHog | Errors + analytics |
| Opslag | Vercel Blob | CV bestanden (PDF/DOCX) |
| Stijl | Tailwind CSS v4 + Radix UI | Design system |
| Validatie | Zod | End-to-end type safety |

## Projectstructuur

```
app/                  # Next.js App Router pagina's
├── api/              # REST routes (22 API route groepen)
├── matching/         # Matching pagina's
├── pipeline/         # Kandidaat pipeline
├── kandidaten/       # Kandidaten overzicht
└── scraper/          # Scraper configuratie

src/
├── ai/               # AI tools en prompts
├── cli/              # CLI (34 commando's)
├── db/               # Drizzle schema (9 tabellen)
├── mcp/              # MCP server (37 tools)
├── schemas/          # Zod validatie schema's
└── services/         # Business logica (28 services)
```

## Conventies

- **Taal**: Nederlandse UI-strings, Engelse code-variabelen
- **API paden**: Nederlands (`/api/scraper-configuraties`, `/api/gezondheid`)
- **Linting**: Biome (niet ESLint/Prettier)
- **Tests**: Vitest met `.test.ts` extensie in `/tests/`
- **Soft deletes**: `deletedAt` timestamp voor GDPR compliance
- **Vector dimensies**: 512 (OpenAI text-embedding-3-small)
